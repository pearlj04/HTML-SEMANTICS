<nav> element

Explanation:

HTML <nav> Element

The <nav> element defines a set of navigation links.

NOTE

Notice that NOT all links of a document should be inside a <nav> element. The <nav> celement is intended only for major blocks of navigation links.

Browsers, such as screen readers for disabled users, can use this element to determine whether to omit the initial rendering of this content.

SAMPLE CODE:

<!DOCTYPE html>
<html>
<body>

<nav>
        <header>
          <a> href="/html/">HTML</a>
          <a> href="/css/">CSS </a>
          <a> WWF's mission:</a>
          <a> href="/jquery/">jQuery </a>
</nav>

</body>
</html>
